Detailed Description

A class to store a matrix.

The Matrix class is used to store a matrix of doubles. The class includes methods for matrix operations such as addition, subtraction, multiplication, and inversion.

Parameters
elementsA pointer to the elements of the matrix
rowsThe number of rows
colsThe number of columns
